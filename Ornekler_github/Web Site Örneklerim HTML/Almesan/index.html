Alüminyum yassı mamul sektöründe, disk ve pul üretiminde lider üretici konumunda olan Almesan Alüminyum, 2009 yılından bu yana sahip olduğu üstün HAZELETT levha bant döküm ve sıcak hadde teknolojisiyle üretim yaparak faaliyetlerini sürdürmektedir.


Kocaeli Arslanbey Organize Sanayi Bölgesi'nde 50.000 m2 kapalı, 100.000m2 açık alana kurulu olan Almesan Alüminyum, SELÇUKLU HOLDİNG kuruluşudur.
